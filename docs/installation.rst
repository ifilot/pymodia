Installation
============

Dependencies
------------

:program:`PyMoDia` is directly dependent on `drawsvg <https://github.com/cduck/drawsvg>`. For instructions to install :program:`drawsvg` it is adviced to follow `their instructions <https://github.com/cduck/drawsvg>`.


PyPI
------------

To install :program:`PyMoDia` systemwide, run::

    sudo pip install pymodia

or to install :program:`PyQInt` only for the current user, run::

    pip install pymodia
